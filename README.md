[JOS Digest](https://jolmos98.github.io/newsletters/).

The daily digest is generated by **OpenClaw** using the **Qwen 3.5 (cloud)** model, scheduled **always** at **14:00** Spain time (**Europe/Madrid**).

## Fetch (fase 1)

Script: `scriptFetchNewsletters.js`

Objetivo:
- Descargar newsletters desde Gmail API.
- Crear estructura `runs/YYYY-MM-DD/raw` y `runs/YYYY-MM-DD/summary`.
- Guardar un raw por newsletter (`raw/<messageId>.md`).
- Registrar estado y metadatos en `runs/YYYY-MM-DD/manifest.jsonl`.

Instalacion:

```bash
npm install
```

Ejemplo de ejecucion manual:

```bash
node scriptFetchNewsletters.js --credentials "C:/ruta/credentials/credentials.json" --token "C:/ruta/credentials/token.json" --days 1 --max-results 100
```

Notas:
- Por defecto usa `is:unread newer_than:1d`.
- Si relanzas, evita duplicados usando `messageId` ya registrado en `manifest.jsonl`.
- No crea `article/` en esta fase (solo `raw/` y `summary/`).
